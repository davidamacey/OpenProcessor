# The vendored end2end patch (src/ultralytics_patches) injects the TensorRT
# EfficientNMS_TRT plugin into YOLO11 ONNX exports. It targets the 8.3.x
# exporter API — ultralytics 8.4 (YOLO26) changed the end2end handling and
# breaks it — so this toolchain is pinned and isolated in its own venv.
# export/export_models.py transparently re-execs into it (see the toolchain
# guard at the top of that script). YOLO26 uses the main environment.
#

# patch. Bump deliberately, never with a range.
